La comunicazione tra component

Component Home (Padre)

  • Qui richiamo i 2 component gmap e tab-bar con le loro proprietà (Input, parentesi quadrate)
  • Quando avviene l'evento (output, parentesi tonde) viene lanciato il metodo updateDimensione che ha come parametro $event, che contiene i paramentri inviati dall'evento.
  • Aggiunta cartella server con dentro un mio json con l'array di oggetti..installato tool json-server che tramite watch crea degli endpoint per ognuno dei json caricati
  • Iniettato il servizio Http nel costruttore
  • A questo punto faccio una chiamata get sull'url creato
  • Questo mi restituisce un Observable, una sorta di stream di informazioni che contiene dati sincroni e asincroni che si può sottoscrivere tramite il metodo subscribe e ricevere i dati non appena sono emessi. (Vedere sidebar per riepilogo)
  • Questi dati devono essere convertiti dal json in oggetti che andranno a popolare il mio array dimensione
  • A questo punto sposto la chiamata in un servizio -> una classe indipendente dalla View che viene definita per svolgere un compito ben preciso
