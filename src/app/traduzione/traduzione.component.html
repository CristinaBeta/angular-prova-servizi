Component traduzione

Titolo e testo sono identificatori che ngx-translate usa per trovare le stringhe di traduzione.
Basta sostituire i testi statici con i riferimenti nel file json.
Ci sono 3 scelte quando si tratta di aggiungere traduzioni:

  • translation pipe — {{'id' | translate}}
  • translation directive — id as attribute value <element [translate]="'id'"></element>
  • translation directive — id as a child <element translate>id</element>

{{ 'demo.title' | translate }}

{{ 'demo.text' | translate }}

demo.text


Traduzione con paramentro

ngx-translate supporta anche i parametri nelle traduzioni. Vengono passati come oggetto, le chiavi possono essere utilizzate nelle stringhe di traduzione

{{ 'demo.greeting' | translate:{'name':'Pippo'} }}
